Agent enrollment

Enroll an existing Ubuntu host with:

curl -fsS http://{{ server_ip }}:{{ http_port }}/agent-src/install.sh | sudo sh -s -- \
  --server http://{{ server_ip }}:{{ web_port }} --token <TOKEN>

Windows (admin PowerShell):

iwr http://{{ server_ip }}:{{ http_port }}/agent-src/windows/install.ps1 -OutFile $env:TEMP\ls.ps1
& $env:TEMP\ls.ps1 -Server http://{{ server_ip }}:{{ web_port }} -Token <TOKEN>
